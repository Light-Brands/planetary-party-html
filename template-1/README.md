#### Typography
- **Primary Font**: Georgia, Times New Roman (serif) - for headings
- **Secondary Font**: Helvetica Neue, Arial (sans-serif) - for body text

#### Key Components
- **Feature Cards**: Elevated cards with hover effects
- **Guild Cards**: Specialized cards with gradient headers
- **Protocol Steps**: Numbered circular badges with descriptions
- **Quote Sections**: Full-width inspirational quote blocks
- **Image-Text Sections**: Flexible grid layouts for content + images
- **Responsive Navigation**: Mobile-friendly menu with smooth animations

### JavaScript Features
- **Mobile Menu**: Responsive hamburger menu with toggle
- **Smooth Scrolling**: Anchor link navigation
- **Scroll Animations**: Intersection Observer for fade-in effects
- **Header Effects**: Shadow changes on scroll
- **Active Nav State**: Current page highlighting

### Responsive Design
- **Mobile-first approach**
- **Breakpoints**:
  - Mobile: < 768px (single column layouts, stacked navigation)
  - Tablet/Desktop: >= 768px (multi-column grids, horizontal navigation)
- **Fluid Typography**: Using clamp() for responsive text sizing
- **Flexible Grids**: CSS Grid with auto-fit for responsive card layouts

## How to Use This Website

### Local Development
1. Simply open `index.html` in a web browser
2. All navigation links will work for completed pages
3. The website is fully static HTML/CSS/JS - no build process required

### Deploying
This website can be deployed to any static hosting service:
- **GitHub Pages**: Push to a repo and enable GitHub Pages
- **Netlify**: Drag and drop the `template-1` folder
- **Vercel**: Connect your repo or drag and drop
- **Traditional Web Hosting**: Upload via FTP to any web server

### Customization

#### Colors
To change the color scheme, edit the CSS variables in `css/styles.css`:
```css
:root {
    --primary-green: #2d5f4f;
    --secondary-green: #4a7c59;
    /* ... etc ... */
}
```

#### Content
All content is in plain HTML and can be edited directly in the `.html` files.

#### Adding Images
1. Copy your images to the `../images/` folder
2. Reference them in HTML using relative paths:
   ```html
   <img src="../images/your-image.jpg" alt="Description" loading="lazy">
   ```
